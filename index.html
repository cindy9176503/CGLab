變換器與標記分水嶺變換三維牙齒影像實例分割

Transformer and Marker-based Watershed Transform for Instance Segmentation of 3D Dental Images

  • 前往網頁 GitHub 前往系統 GitHub

摘要

本研究提出了基於變換器與標記分水嶺變換之方法, 以解決從三維錐狀射束電腦斷層掃描 (CBCT) 影像進行準確地牙齒實例分割之問題。 由於 CBCT 影像具有牙齒結構複雜、牙齒周圍邊界不清晰以及金屬偽影等因素，手動進行牙齒實例分割是一項耗時且具有挑戰性的任務。 為了解決這個問題，首先，本研究提出了 CSWin UNETR 之編碼器解碼器網路，由 CSWin Transformer 與 UNETR 組成。並將 CSWin UNETR 組成雙分支網路 以同時預測牙齒區域與中心。最後，透過標記分水嶺變換實現牙齒實例分割。實驗結果表明，提出之方法能於金屬植體、智齒與缺牙的影像中有效地分割單個牙齒。 此外，本研究所提出之 CSWin UNETR 在牙齒區域分割任務中，取得之 Dice 相似係數為 0.9640，95% Hausdorff 距離為 1.0414 mm， 其定量分析結果皆優於 Swin UNETR 現有之優秀架構。其結果表明基於變換器與標記分水嶺變換之方法能解決三維牙齒影像實例分割之問題。
